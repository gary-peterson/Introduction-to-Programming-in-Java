Problems Part 1 of 2

It is advised to work through the examples in the chapter before doing these problems.


Take three classes that you have previously defined (e.g. in previous assignment) or if you prefer, you may come up with completely new classes.

Your three classes should not include the examples Rectangle, Circle, Student or Greeter.

For each class, do the following:
  • Declare one or more instance variables
  • Define one or more constructors
  • Define one or more instance methods. These methods should not be constructors and not static methods.
  • Declare and use one or more local variables -- these would be part of instance method(s).
  • Declare and use one or more method parameters -- these would be part of instance method(s).

Hints

Before submitting JAVA files, make sure they compile without errors.

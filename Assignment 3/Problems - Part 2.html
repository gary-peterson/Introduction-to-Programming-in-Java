Problems Part 2 of 2

For this part of the problems, you will write one "README.TXT" file.

The one "README.TXT" file should include for each of your three JAVA files instructions on how to use each class. The instructions should include how to construct two different objects for the class, and how to print those two objects. And from the print output the user should be able to see the difference between the objects.

Here is an example "README.TXT" for the Rectangle class in the examples:

README.TXT

-------------------------------------
Class: Rectangle
User Instructions:

Rectangle rec1, rec2;
rec1 = new Rectangle(8, 7);
System.out.println("Rectangle #1 is: " + rec1);
rec2 = new Rectangle(5, 1);
System.out.println("Rectangle #2 is: " + rec2);

-------------------------------------


Your "README.TXT" may be similar to the example above (except you would have three sections showing instructions for each of your three classes).

Note that if we follow the instructions in the example "README.TXT" file above, and try this in a JAVA file using the Rectangle class, our output is something like:
Rec-1 -- width=8 height=7 -- area=56 Perimeter=30
Rec-2 -- width=5 height=1 -- area=5 Perimeter=12
Press any key to continue . . .


We can easily tell the difference between the two objects.
